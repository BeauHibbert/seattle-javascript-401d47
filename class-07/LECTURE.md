## Bearer Auth

- JWT - JSON Web Token

- token create to represent your identity
- we will create a JWT upon signin
- anatomy:  aaaaaaaa.bbbbbbbbb.ccccccccc
  - a: header
  - b: payload: data content aka (username, password) 
  - c: signature: SECRET

HOW is JWT secure?   ANSWER:  the signature, or SECRET
